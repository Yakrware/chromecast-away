Receiver examples

Default Receiver

Play audio, video, and photos using the chromecast's default receiver. You do not need to register an app id for this receiver.

Stylized Media Receiver

Play audio, video, and photos using the chromecast's default media receiver but provide a link to a stylesheet to decorate it. You do need to register an app id for this receiver and the stylesheet needs to be loaded from a https connection.

Custom Receiver

Load a url containing HTML, CSS, and JS onto your chromecast and communicate with it via message passing. You do need to register an app id for this receiver and provide a namespace for the messages.
